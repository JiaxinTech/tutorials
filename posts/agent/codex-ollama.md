### Ollama常用命令：

| **命令** | **说明** |
| --- | --- |
| **`ollama run <模型名称>`** | **启动模型并直接进入聊天窗口。** 如果本地没有该模型，会自动从云端下载。 |
| **`ollama pull <模型名称>`** | **只把模型下载到本地电脑。** 不会自动启动聊天，适合半夜挂机下载大模型。 |
| **`ollama serve`** | **手动启动 Ollama 的后台服务器服务。**  |
| **`ollama list`** | **列出当前电脑硬盘里已经下载好的所有本地大模型名字和大小** |
| **`ollama rm <模型名称>`** | **彻底删除某个本地模型** |
| **`ollama ps`** | **查看当前后台有哪些模型正在运行。** 以及它们分别占用了多少内存和显存。 |
| **`ollama show <模型名称>`** | **查户口命令。** 详细显示某个模型的底层参数、架构以及系统提示词（Prompt）。 |
| **`ollama launch codex`** | 用本地模型启动 Codex |
| **`OLLAMA_MAX_LOADED_MODELS=1`**  | **允许内存里同时常驻 1个模型。** 比如你想让 DeepSeek 和 Qwen 同时在后台待命，切着工具用，不用每次都花几秒钟重新加载。 |
| **`codex -oss -m <模型名称>`** | 以指定模型启动codex |

### 模型卸载流程

```
                ┌──────────────────────┐
                │   API / Codex 请求    │
                └──────────┬───────────┘
                           │
                           ▼
                ┌──────────────────────┐
                │   模型是否已加载？    │
                └──────────┬───────────┘
                           │
              ┌────────────┴────────────┐
              │                         │
            否（No）                  是（Yes）
              │                         │
              ▼                         ▼
   ┌───────────────────┐     ┌─────────────────────┐
   │   加载模型到内存   │     │   直接复用已加载模型  │
   └─────────┬─────────┘     └──────────┬──────────┘
             │                          │
             ▼                          ▼
   ┌───────────────────┐     ┌─────────────────────┐
   │  进入推理（running）│     │  进入推理（running） │
   └─────────┬─────────┘     └──────────┬──────────┘
             │                          │
             ▼                          ▼
        返回结果                   返回结果
             │
             ▼
   ┌──────────────────────────────┐
   │  是否 keep_alive 计时开始？   │
   └──────────────┬───────────────┘
                  │
        ┌─────────┴─────────┐
        │                   │
      是（默认）          0（立即卸载）
        │                   │
        ▼                   ▼
┌──────────────────┐   ┌──────────────────┐
│ 模型继续驻留内存  │   │ 立即释放模型内存  │
└─────────┬────────┘   └──────────────────┘
          │
          ▼
   ┌──────────────────────┐
   │ 进入“cached loaded”   │  ← 你在 ollama ps 看到的
   └─────────┬────────────┘
             │
             ▼
   ┌──────────────────────┐
   │ LRU / max_loaded_models│
   │ 决定是否被踢掉        │
   └─────────┬────────────┘
             │
     超过限制？
        │
   ┌────┴────┐
   │         │
  是         否
   │         │
   ▼         ▼
┌────────┐  保留
│卸载旧模型│
└────────┘
```

```bash
#内存里最多只能“放 1 个模型”
export OLLAMA_MAX_LOADED_MODELS=1

#模型用完立即卸载
export OLLAMA_KEEP_ALIVE=0

#同一个模型同时能处理多少个请求
export OLLAMA_NUM_PARALLEL=1
```
